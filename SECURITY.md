# Security Policy

## Supported Versions

Only the latest version of AIOps Hub is currently supported with security updates.

| Version | Supported |
| ------- | --------- |
| Main    | ✅ Yes     |
| < 1.0.0 | ❌ No      |

## Reporting a Vulnerability

We take the security of AIOps Hub seriously. If you find any security vulnerabilities, please do not report them publicly via GitHub issues. Instead, follow these steps:

1. Send an email directly to `81868595+sadiab47@users.noreply.github.com` outlining the details of the vulnerability.
2. Include steps to reproduce the issue, sample code, or payloads where possible.
3. We will acknowledge receipt of your report within 48 hours and work with you to resolve and patch the issue.

Please allow us reasonable time to address and patch the issue before making it public.
